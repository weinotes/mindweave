📖 产品简介

MindWeave 是一款基于 Ollama 的本地 AI 对话工作台。无需联网，所有对话数据存储在本地，支持多用户协作，适合办公室或团队在局域网内共享使用。

💡 核心理念

你的数据永远存储在自己的电脑上，不经过任何第三方服务器。大模型运行在本地硬件，零网络延迟，零数据泄露风险。

适用场景

  • 办公室内部 AI 助手（无需互联网）
  • 需要数据完全私有的工作场景
  • 团队共享使用，降低每人一台设备的模型部署成本
  • 日常文档写作、代码辅助、资料整理

⚙️ 安装配置

环境要求

项目要求说明
操作系统macOS 12+ / LinuxWindows 建议 WSL2
内存≥ 8GB16GB 体验更佳
磁盘≥ 10GB存放模型文件
PHP≥ 8.1Laravel 11 运行要求
Composer≥ 2.0PHP 包管理器
Ollama≥ 0.1大模型运行时

第一步：安装 Ollama

Ollama 是本地大模型的运行时引擎，负责下载和管理模型。

$ brew install ollama

第二步：下载模型

根据你的硬件配置选择合适的模型。模型越大，效果越好，但对内存要求越高。

模型大小内存要求适用场景
qwen2.5:3b1.8GB≥ 4GB快速问答、轻量辅助
qwen2.5:7b4.4GB≥ 8GB日常办公、文档写作
qwen2.5:14b8.2GB≥ 16GB代码辅助、深度分析
poet-qwen:latest~2GB≥ 6GB中文对话优化版
$ ollama pull qwen2.5:3b # 推荐入门
$ ollama pull qwen2.5:7b # 进阶配置

第三步：启动 Ollama 服务

$ ollama serve

Ollama 默认监听 http://localhost:11434，保持运行即可。

第四步：安装 MindWeave

$ cd ~/mindweave $ composer install $ php artisan serve --host=0.0.0.0 --port=3456
🎉 访问地址

本地访问：http://localhost:3456
局域网访问（见「内网部署」章节）：http://你的IP:3456

🚀 快速上手

1

首次登录

首次访问时无需输入密码，直接以「访客」身份进入。可在设置中创建专属账号。

2

选择大模型

在顶部「模型」下拉框中选择已下载的模型。不同模型擅长不同场景，可按需切换。

3

开始对话

在底部输入框输入问题，按 Enter 或点击「发送」。支持 Markdown 语法，代码块自动高亮显示。

4

新建对话

点击左侧「+ 新建对话」开启新会话。每个会话自动保存在侧边栏，随时切换。

🔧 功能说明

对话管理

  • 新建对话 — 侧边栏点击「+ 新建对话」，自动生成新会话
  • 切换会话 — 点击侧边栏历史会话，查看之前对话内容
  • 删除会话 — 鼠标悬停会话，右侧出现「删除」按钮
  • 清空当前 — 点击顶部的「🗑️ 清空」按钮，清除当前对话内容

对话导出

  • 导出 MD — 将当前对话导出为 Markdown 格式，适合直接粘贴到文档
  • 导出 JSON — 导出为 JSON 格式，适合程序处理或备份

Markdown 支持

助手回复支持完整 Markdown 渲染，包括：

  • 代码块（高亮显示、等宽字体）
  • 行内代码
  • 粗体、斜体
  • 引用块
  • 有序/无序列表
  • 表格
  • 标题分级

模型选择

顶部模型选择器支持切换不同的大模型。切换模型后，后续对话将使用新模型，已有的历史消息不受影响。建议根据任务类型选择合适的模型：

  • 日常问答、写作 → 小模型（qwen2.5:3b）响应快
  • 代码辅助、复杂分析 → 大模型（qwen2.5:7b+）效果更好

👥 多用户管理

MindWeave 支持多账号系统，每个用户拥有独立的会话存储空间，数据完全隔离。

创建用户

1

进入设置

点击侧边栏底部的「用户管理」按钮

2

添加账号

在用户管理界面输入用户名，点击「创建」

3

设置密码（可选）

点击用户旁边的「改密」按钮设置独立密码。不设置密码则无需验证直接进入。

🔐 访客模式

访客账号（guest）无需密码，任何人可直接进入。适合公共设备或快速体验。访客的对话数据同样独立存储。

账号管理

  • 修改密码 — 点击「改密」，输入新密码，留空则清除密码
  • 删除用户 — 点击「删除」，该用户的所有对话记录将同步删除
  • 访客账号 — 系统内置，不可删除

🌐 内网部署

如果需要让局域网内其他同事访问 MindWeave，需要将服务暴露到局域网。

查找本机 IP

$ ipconfig getifaddr en0 192.168.1.100

启动服务（局域网模式）

$ cd ~/mindweave $ php artisan serve --host=0.0.0.0 --port=3456

同事访问

在同一局域网下，打开浏览器访问：

$ http://192.168.1.100:3456
⚠️ 安全提示

局域网模式下，同一网络内的设备均可访问。建议设置账号密码保护（见「多用户管理」），防止未经授权的访问。

开机自启（可选）

如需 MindWeave 在 Mac 启动时自动运行，可使用 launchd 或 Homebrew Services：

$ brew services run laravel-valet # 或配置 launchd

❓ 常见问题

Q: 页面打不开，显示空白？

请确认两个服务都在运行：ollama serve（Ollama）和 php artisan serve（MindWeave）。另外确认端口 3456 未被其他程序占用。

Q: 发送消息后没有回复？

检查 Ollama 是否正常运行：curl http://localhost:11434。如果返回版本信息则正常。如果模型未下载，先执行 ollama pull 模型名。

Q: 回复速度很慢？

模型越大，生成速度越慢。建议使用较小的模型（如 qwen2.5:3b）进行日常对话。另外，确保 Mac 有足够的可用内存。

Q: 对话数据存在哪里？

所有对话数据存储在 mindweave-data/sessions/ 目录下，每个用户的数据在各自的用户名文件夹下。备份时只需备份整个 mindweave-data 文件夹即可。

Q: 如何更换大模型？

先下载新模型：ollama pull qwen2.5:7b，然后在 MindWeave 顶部模型选择器中切换即可。

Q: 能远程访问吗？

可以。推荐使用 Tailscale 将多台设备组成虚拟局域网，然后按「内网部署」方式访问。
